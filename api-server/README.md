# api.sleeplog-app.com

> API for managing users and sessions for sleeplog-app.

## Build Setup

``` bash
# install dependencies
go get dependency-name

# update Godeps dependencies
godep save

# serve at localhost:8089
go run main.go

# run unit tests
go test
```

## Deploy to Heroku

``` bash
# Create heroku app
heroku create

# Deploy
git push heroku master
