# api.enoplay.com

> API for managing games, users and sessions for Enoplay.

## Build Setup

``` bash
# install dependencies
go get dependency-name

# update Godeps dependencies
godep save

# serve at localhost:8089
go run main.go

# run unit tests
go test
```

## Deploy to Heroku

``` bash
# Create heroku app
heroku create

# Deploy
git push heroku master
